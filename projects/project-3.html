Attendix

"Attendix" is a SaaS mobile app designed for managing employee schedules and tasks. It features task cards that display today's schedule, which can include shifts or project tasks. The app aims to enhance productivity and organization within teams.

Developed with Flutter, Attendix ensures a smooth user experience with color-coded task differentiation and a consistent design across all screens. The backend is powered by Laravel to handle data efficiently and provide reliable performance.

Attendix helps managers keep track of employee activities and streamline task management with an intuitive interface.

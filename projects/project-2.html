MediBook

"MediBook" is a SaaS mobile application designed to streamline appointment scheduling and medical record management. Developed using Flutter, the app offers a user-friendly interface and features such as appointment reminders, medical history tracking, and personalized doctor recommendations.

With a focus on usability, the application integrates advanced features to enhance user experience, including real-time notifications and a secure backend managed with Laravel to ensure data integrity and performance.

MediBook aims to simplify healthcare management for both patients and doctors by providing an efficient and accessible platform.

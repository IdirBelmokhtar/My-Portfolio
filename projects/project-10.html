Tic-Tac-Toe

"Tic-Tac-Toe" is a simple yet entertaining mobile app that recreates the classic game of Tic-Tac-Toe. It allows users to play against AI or another player, featuring an intuitive and responsive interface.

The app is designed for smooth performance and an engaging user experience. It uses Firebase to manage game data and user interactions efficiently. The source code is available on GitHub for those interested in exploring or contributing to the project.
